# task-tracker-app

C# ve WinForms kullanarak geliştirilmiş bir masaüstü görev takip uygulaması.

## Özellikler
- Görev ekleme, silme ve güncelleme
- Tamamlanan / bekleyen görevleri listeleme
- Kategori ve son tarih yönetimi

## Teknolojiler
- C#
- WinForms
- Entity Framework
- SQL Server

## Kurulum
1. Projeyi klonla
2. Visual Studio ile aç
3. Veritabanı bağlantısını kontrol et
4. Çalıştır
